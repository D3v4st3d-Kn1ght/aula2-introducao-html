!DOCTYPE
*Informa a versão do HTML
html
*Inicia o documento HTML
head
*Onde fica tudo que não aparece na página, configurações, ajustes, etc.
body
*Representa o conteúdo visível de uma página, incluindo textos, imagens, vídeos e links
h1
*Marca um título
p
*Marca um início de parágrafo
style
*O processo de aplicar CSS (Cascading Style Sheets) para definir a aparência de elementos, como cores, layouts e fontes
script
*É usada para incorporar ou referenciar código JavaScript em páginas web, permitindo interatividade.
img
*Adiciona o endereço de uma imagem no servidor.
br
*Informa uma quebra de linha.
ul
*Determina uma lista não ordenada.
ol
*Determina uma lista ordenada.
li
*Determina um elemento da lista.
dl
*Determina uma lista de definição.
dt
*Identifica um termo na lista de definição
dd
*Definição mais completa do termo informado por dd
